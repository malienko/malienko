# Marketplace Images

Place your marketplace item images in this folder with the following filenames:

- `table-legs.jpg` - Table legs product image
- `office-chair.jpg` - Office chair product image
- `gaming-pc.jpg` - Gaming PC product image

Images should be:
- Format: JPG or PNG
- Recommended size: 300x300px minimum
- Aspect ratio: Square (1:1) works best for display
